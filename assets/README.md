# Profile Assets

This folder contains the local visual assets used by the GitHub profile README.

## Files

- `banner.gif` - Cozy pixel art desk banner (1280×720). Imported manually; shows a nighttime workspace with a cat, monitor, and city view.
- `banner-classic.svg` - Original static SVG artwork (soft cinematic research theme). Kept as a fallback reference.
- `neon-divider.gif` - Deprecated cyberpunk divider (no longer used in README).
- `neon-divider.png` - Static poster frame from the deprecated divider animation.
- `pets/` - Section pixel cats used in the README. See below.
- `section-divider.gif` - Deprecated previous divider asset.
- `wave-divider.svg` - Source SVG for the deprecated divider.
- `wave-divider.png` - Deprecated; kept for reference.

## Banner

The README uses `banner.gif` as a self-hosted cozy banner at full width (`width="100%"`). It is **not** regenerated by `npm run generate:banner`; update it by replacing the file directly (see `scripts/import-cozy-assets.mjs` for the import workflow).

The legacy cyberpunk banner generator still lives in `scripts/generate-banner/` if you want to experiment, but it will overwrite `assets/banner.gif` if run.

### Legacy skristi cats

Older section cats (`cat-about.gif`, `cat-compass.gif`, `cat-toolkit.gif`, `cat-focus.gif`, `cat-snapshot.gif`) remain in `pets/` but are no longer referenced. They were built from **[Gray cat asset pack](https://opengameart.org/content/gray-cat-asset-pack)** by Krystsina Staselovich (skristi), CC-BY. Source PNGs live in `pets/source/skristi-gray-cat/64x64/`. See `pets/source/ATTRIBUTION.md`.

### Regenerating legacy pets

Built from source in `scripts/generate-pet/`:

```bash
npm install
npm run generate:pet
```

This slices the 64×64 source strips, removes black backgrounds, and writes transparent GIFs to `assets/pets/` (128×128 output, 2× scale).

### Importing cozy cats

To re-import the cozy banner and cat GIFs from Downloads:

```bash
node scripts/import-cozy-assets.mjs
```

### Processing toolkit cat

To regenerate the pixelated headphones cat from Downloads:

```bash
npm run process:toolkit-cat
```

This crops the source GIF, pixelates it to 112px width, and writes `assets/pets/cat-headphones.gif`.
